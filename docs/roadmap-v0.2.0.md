# Roadmap: v0.2.0

## 목표

- README/문서 정합성 마무리
- 앱 흐름을 보여 주는 실제 데모 보강
- UI 테스트 신뢰도 확대
- 릴리스 확장성 개선

## 범위

### 1) 사용자 경험 다듬기

- Article 상세 화면 입력 UX 보강
  - 저장/수정 중 필수 입력값 가드
  - 사용자 피드백 메시지 개선

### 2) 데모/온보딩 개선

- [x] `docs/images/demo.gif`를 실제 앱 동작으로 교체
  - 목록 → 상세 편집 → 수정 → 저장 → 목록 반영 흐름

### 3) 테스트 강화

- 현재 Compose UI smoke test를 확장
  - Back 동작에서 변경 이력 미반영 시나리오
- 잘못된 선택값 처리 테스트(guard/예외 경로 문서화)

### 4) 릴리스/CI 개선

- v0.2 릴리스 태그 정책 정리 및 문서 보강
- `v*` 태그 릴리스 가이드 일관화
- 필요 시 Android CI에서 Gradle cache/워크플로우 분리 단계 검토

## 완료 조건 (Definition of Done)

- 문서의 남은 `v0.2` 체크리스트 해소
- 실제 데모 이미지 반영
- 핵심 UI 플로우 테스트 신뢰도 향상
- v0.2 릴리스 노트 패턴 정착
